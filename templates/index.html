About Orriona GPS Tracker

Orriona is a DIY GPS tracking solution designed for precision and real-time location tracking. The tracker integrates the Wemos D1 microcontroller, which is based on the ESP8266 Wi-Fi module, and the NEO-6M GPS module. This system is ideal for applications requiring continuous GPS tracking with embedded data logging capabilities.

Wemos D1 and NEO-6M GPS Tracker Overview

The Wemos D1 board serves as the core microcontroller. It's equipped with the ESP8266 chip, which provides Wi-Fi connectivity. The NEO-6M GPS module receives GPS signals from satellites, processes them, and communicates the position data to the Wemos D1. The Wemos D1 then sends the location data via Wi-Fi to the server or logs it for later use.

How Does the NEO-6M GPS Module Work?

The NEO-6M GPS module works by receiving signals from the GPS satellites orbiting Earth. Once it gets a signal from at least three satellites, it can calculate its latitude, longitude, and altitude with great accuracy. This information is sent to the Wemos D1, which acts as the interface between the GPS module and your network, enabling real-time tracking.
